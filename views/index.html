Microsserviço de carimbo de data/hora

Use esta API para converter sequências de datas em carimbos de data/hora UNIX e formato UTC.

Instruções

Você pode fazer solicitações GET para os seguintes endpoints:

  • /api/:date - where :date is a date string or UNIX timestamp
  • /api - returns the current date in both UNIX and UTC formats

Examples:

  • /api/2024-09-24
  • /api/1450137600000
  • /api (current date)
